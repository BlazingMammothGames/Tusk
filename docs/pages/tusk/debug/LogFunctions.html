    package tusk.debug

    enum

    tusk.debug.LogFunctions

    Generally not used by user-facing code, this enum is for tracking the type of log that is being written.

    Values

    Fatal

    Show-stopping error

    Error

    Crashes / exceptions

    Warn

    Incorrect behaviour but can continue

    Info

    Indicates correct behaviour

    Debug

    Behind-the-scenes poking around

    Trace

    Full object tracing for deep inspection
