    package tusk.macros

    class

    AutoSerializer

    For automatically adding in variable and property serialization as a build macro.

    NOTE: only public, non-static variables and properties will be serialized!

    Static Methods

    apply ( ) : Dynamic static

    Run the auto serializer on the class. Used automatically on all components with:

    @:autoBuild(tusk.macros.AutoSerializer.apply())

    Returns

    an array of the fields in the class
