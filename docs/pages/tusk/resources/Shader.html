    new ( path : String , ? vertSrc : String , ? fragSrc : String )

    Construct a new shader with the given vertex and fragment sources. Note: you must compile this to use it!

    Parameters

    path

    an identifier for the shader

    ?vertSrc

    a string containing the vertex shader source

    ?fragSrc

    a string containing the fragment shader source

    Properties

    The fragment shader source After changing this, don't forget to recompile!

    program: GLProgram read only

    The program that this shader compiled into

    vertSrc: String

    The vertex shader source After changing this, don't forget to recompile!
